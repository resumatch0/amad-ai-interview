# 🎤 Amad – AI Interview System (Core Engine)

Amad (أمد) is an AI-powered interview engine that simulates structured interviews using voice and large language models.

---

## Overview

This repository contains the **core logic and AI pipeline** behind the Amad interview system.

It focuses on:

* Interview flow design
* AI-driven questioning
* Candidate response evaluation
* Score generation

---

## What’s Implemented

* Voice-based interaction logic (record + transcribe)
* AI interviewer (Qwen)
* Evaluation model (Mistral)
* Multi-step interview flow
* Final scorecard generation

---

## Tech Stack

* Python
* HuggingFace Transformers
* Whisper (Speech-to-Text)
* gTTS (Text-to-Speech)
* LlamaIndex (Embeddings / Retrieval)
